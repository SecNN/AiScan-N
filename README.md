## 在Kali Linux 运行AiScan-N-Kali-Server 开启服务

```
sudo ./AiScan-N-Kali-Server.bin -token 123456
```

🚨token用于API接口鉴权的Bearer令牌，未设置 token，将受到被恶意漏洞利用的高危风险 ⚠️ 建议设置强密码型令牌提高安全性。
